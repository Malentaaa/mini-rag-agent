# О проекте

Mini RAG Agent — локальное AI-приложение для поиска информации внутри PDF документов.

Система:

* индексирует PDF файлы
* создаёт эмбеддинги
* выполняет смысловой поиск информации
* находит наиболее релевантные страницы
* отображает оригинальные страницы PDF
* работает полностью локально внутри Docker

Приложение использует поиск по смысловой близости и семантические векторные представления текста для поиска информации.

# Embedding модель

В проекте используется:

```text id="8-vesm0"
BAAI/bge-m3
```

Почему была выбрана именно эта модель:

* многоязычные эмбеддинги
* высокое качество поиска релевантной информации
* хорошее понимание смысловых связей в тексте
* хорошо работает с длинными запросами
* подходит для RAG систем

---

# Vector Search

Для vector search используется:

```text id="7-vesm0"
FAISS IndexFlatIP
```

Перед индексацией эмбеддинги нормализуются:

```python id="6-vesm0"
faiss.normalize_L2(embeddings)
```

Это позволяет выполнять поиск по смысловой близости (cosine similarity) через внутреннее скалярное произведение векторов (inner product).

# Preview страниц

В виду особенностей строения PDF, вместо отображения “сломанных” chunk-фрагментов система показывает оригинальные страницы PDF.

Когда пользователь нажимает:

```text id="5-vesm0"
Просмотреть страницу
```

backend:

1. открывает PDF
2. рендерит нужную страницу
3. создаёт PNG в памяти
4. отправляет изображение в браузер

Это позволяет сохранить:

* формулы
* таблицы
* изображения
* layout документа
* оригинальное форматирование

# Автоматическое обновление индекса

При запуске приложение проверяет:

* список PDF файлов
* размер файлов
* дату изменения
* embedding модель

Если что-то изменилось:

```text id="4-vesm0"
индекс автоматически пересобирается
```

Metadata хранится в:

```text id="3-vesm0"
storage/indexes/index_meta.json
```

# Запуск через Docker

## 1. Клонировать репозиторий

```bash id="1-vesm0"
git clone <your_repo>
cd mini-rag-agent
```

---

## 2. Добавить PDF документы

Поместить PDF файлы в:

```text id="0-vesm0"
data/docs
```

---

## 3. Собрать Docker контейнер

```bash id="9vjlwm"
docker compose build
```

---

## 4. Запустить приложение

```bash id="8vjlwm"
docker compose up
```

---

## 5. Открыть браузер

```text id="7vjlwm"
http://127.0.0.1:8000
```
